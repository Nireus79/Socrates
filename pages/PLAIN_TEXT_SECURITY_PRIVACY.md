FAQ - SECURITY AND PRIVACY

Is my API key safe?

Your key is never stored by us. Only transmitted to Claude API. Keep it secret like a password. Rotate if exposed.

Can you train AI models on my code?

No! We explicitly don't train models on user data. This is a core promise and contractual obligation.

Is my data shared with third parties?

No. We use AWS (infrastructure) and Stripe (payments), but no code data is shared with anyone. We don't sell data.

Is Socrates GDPR compliant?

Yes! EU data residency available. Data deletion implemented. Full GDPR support. Data Processing Agreement available.

What's your security track record?

No known breaches. Regular security audits. Vulnerability disclosure program. SOC 2 coming Q2 2026.

How is data encrypted?

TLS 1.3 in transit. AES-256 at rest. Keys rotated every 90 days. Optional end-to-end encryption for Enterprise.

Can I export all my data?

Yes! Anytime. Formats: JSON, CSV, Markdown, PDF. No restrictions on use. Available immediately.

What happens if you get hacked?

We notify users within 24 hours. Transparency in incident communication. We have incident response procedures. Insurance coverage in place.

How long do you keep backups?

Daily backups for 30 days. Weekly backups for 90 days. Monthly archival for 1 year. Tested restore procedures.

Can I use Socrates with sensitive data?

For highly sensitive data: Use Enterprise self-hosted option. For standard business data: Yes, safe to use. Encrypt locally for extra protection.

Does Socrates meet healthcare compliance?

Not currently HIPAA compliant. Coming with Enterprise offering. Contact sales@socrates-ai.com for healthcare needs.

What about compliance for government projects?

Contact sales@socrates-ai.com. We can discuss:
- FedRAMP considerations
- Government data requirements
- Custom compliance agreements

Do you use your own cloud or shared?

We use AWS cloud infrastructure. You don't share storage with others. Enterprise can use self-hosted or private cloud.

How often do you audit security?

Monthly scans. Quarterly penetration testing. Annual third-party audits. Continuous vulnerability assessment. SOC 2 audit coming Q2 2026.
