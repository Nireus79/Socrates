FREQUENTLY ASKED QUESTIONS ABOUT PRICING

What are Claude API costs?

Socrates uses Claude API for code generation and dialogue. You pay Anthropic directly. Typical costs are $1-5 per project. See pricing at https://www.anthropic.com/pricing

Do I need to pay for Socrates AND API costs?

Yes. Socrates subscription covers the platform. Claude API costs are separate and reflect actual usage.

Example:
- Pro tier: $15/month
- API costs per project: $1-5
- Total: $15 + (API costs per project)

Can I upgrade or downgrade anytime?

Yes! No lock-in contracts. Change your plan anytime. Pro-rated charges apply for mid-month changes.

Is there a free trial?

Yes! The Free tier is a full-featured trial. No credit card required.

Do you offer discounts?

Yes! We offer:
- 50% student discount (with .edu email)
- Free Pro tier for open source projects (100+ stars)
- Custom academic pricing for institutions
- Volume discounts for enterprises

How do I monitor API costs?

Monitor in Settings → Token Usage. See:
- Questions asked this month
- Dialogue tokens used
- Code generation tokens used
- Total estimated cost
- Remaining API budget (if applicable)

What if I exceed my API limit?

Free tier: 100 requests/day limit
Basic tier: 1,000 requests/day limit
Pro tier: 10,000 requests/day limit

If you hit the limit, you can:
1. Wait until the next day for reset
2. Upgrade to higher tier for more requests
3. Purchase additional tokens (Enterprise)

Can I change my billing period?

Monthly billing available now. Annual billing (with discount) coming soon.

What payment methods do you accept?

- Credit/Debit cards (Visa, Mastercard, American Express)
- PayPal
- Bank transfer (for enterprise customers)

All payments processed securely by Stripe.

Is there a refund policy?

Pro Tier: 30-day money-back guarantee
Basic/Free: No refund needed (no payment)

If unsatisfied, contact support@socrates-ai.com within 30 days for full refund.

What happens if I cancel?

Your account is kept for 30 days. You can:
- Reactivate anytime within 30 days
- Export all your data (specifications, projects)
- No lock-in or cancellation fees

Data is deleted after 30 days per GDPR.

How does the Free tier work with API costs?

You still pay Claude API costs. Free tier subscription is $0, but you pay for actual API usage as you go.

Example:
- Free tier: $0/month platform fee
- API usage: $3 (for 3 projects)
- Total cost: $3
