## Frequently Asked Pricing Questions

### Q: What are API costs?
**A:** Socrates uses Claude API from Anthropic. You pay Anthropic directly for API usage:
- Haiku model: ~$0.01-0.05 per request
- Typical project: 20-30 questions = $1-5 total
- Code generation: $0.50-2.00 per project
- We don't add markup—pay Anthropic's rate directly

We bill your API usage separately. Your Socrates subscription fee never includes API costs.

### Q: Can I upgrade/downgrade anytime?
**A:** Yes!
- Upgrade anytime (prorated billing)
- Downgrade anytime (takes effect next billing cycle)
- Cancel anytime (no lock-in contracts for monthly plans)

### Q: What happens if I exceed storage?
**A:**
- Free tier: You'll be notified at 90%, then 100%. Create a new project or upgrade.
- Paid tiers: No overage charges. Upgrade to next tier for more storage.

### Q: Do I get a discount for annual billing?
**A:**
- Monthly: $5/month (Basic), $15/month (Pro)
- Annual: 15% discount → $51/year (Basic), $153/year (Pro)
- Contact sales for multi-year discounts

### Q: Is there a free trial for Pro?
**A:**
- Free tier is a full-featured trial with storage limits
- Upgrade to Pro anytime, no credit card required for Free tier
- 30-day money-back guarantee on Pro tier

### Q: Can multiple people use one account?
**A:**
- Free/Basic: Personal use only (1 account)
- Pro: Add up to 5 team members
- Enterprise: Unlimited team members
- Each person gets their own login and permissions

### Q: What's included in "Support"?
**A:**
- Free: Community forums (Discord, GitHub)
- Basic: Email support within 48 hours
- Pro: Email support within 24 hours
- Enterprise: Dedicated Slack channel, 2-hour response

### Q: Do you offer student/nonprofit discounts?
**A:**
Yes! Contact [hello@socrates-ai.com](mailto:hello@socrates-ai.com)
- Students: 50% off Pro tier
- Nonprofits: 50% off Pro tier
- Open source projects: Free Pro tier
- Educational institutions: Custom pricing

### Q: What if I'm not happy?
**A:**
- Free tier: No commitment
- Pro tier: 30-day money-back guarantee
- Enterprise: Custom terms available

### Q: Can I self-host?
**A:**
- Free/Basic: Cloud-only
- Pro: Cloud-only
- Enterprise: Self-hosted option available
- Open source: Download and run locally (MIT license)
