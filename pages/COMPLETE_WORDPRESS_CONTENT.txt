INSTALLATION - Socrates AI

Choose your platform and follow the steps. All options are free to try.

QUICK START

Windows: Download and run installer
macOS/Linux: git clone https://github.com/Nireus79/Socrates.git
Docker: docker run -it ghcr.io/nireus79/socrates:latest

WINDOWS

Option A: Download Executable (Easiest)

Prerequisites: Windows 10/11, 500 MB free disk space

Steps:
1. Download from https://github.com/Nireus79/Socrates/releases/latest
   File: socrates-v1.3.0-windows.exe (280 MB)
2. Run the installer
3. Follow the installation wizard
4. Click "Launch Socrates"
5. Browser opens automatically

Set API Key:
1. Go to https://console.anthropic.com/
2. Create API key
3. In Socrates Settings, paste API key
4. Save

Benefits: No programming knowledge needed, one-click installation, automatic updates, desktop shortcut

System Requirements: Windows 10/11, 4 GB RAM minimum, 500 MB disk, internet required

Option B: Clone from GitHub

Prerequisites: Git and Python 3.9+ installed

Steps:
1. git clone https://github.com/Nireus79/Socrates.git
2. cd Socrates
3. python -m venv venv
4. venv\Scripts\activate
5. pip install -r requirements.txt
6. set CLAUDE_API_KEY=your-key-here
7. socrates /project create

Benefits: Full source code, can customize, easy to contribute

MACOS

Option A: Homebrew (Easiest)

Prerequisites: Homebrew installed (https://brew.sh/)

Steps:
1. brew tap nireus79/socrates
2. brew install socrates
3. socrates /project create

Browser opens at http://localhost:8000

Option B: Clone from GitHub

Prerequisites: Git and Python 3.9+

DOCKER

Prerequisites: Docker installed and running

Steps:
1. docker pull ghcr.io/nireus79/socrates:latest
2. docker run -it -p 8000:8000 -e CLAUDE_API_KEY=your-key-here ghcr.io/nireus79/socrates:latest

Access at: http://localhost:8000

With Persistent Data:

docker run -it -p 8000:8000 -v socrates-data:/app/data -e CLAUDE_API_KEY=your-key-here ghcr.io/nireus79/socrates:latest

Docker Compose:

Create docker-compose.yml:

version: '3.8'
services:
  socrates:
    image: ghcr.io/nireus79/socrates:latest
    ports:
      - "8000:8000"
    environment:
      - CLAUDE_API_KEY=your-key-here
    volumes:
      - socrates-data:/app/data
    restart: unless-stopped

volumes:
  socrates-data:

Run: docker-compose up -d

TROUBLESHOOTING

Port 8000 already in use:
set SOCRATES_PORT=8001 (Windows)
export SOCRATES_PORT=8001 (macOS/Linux)

API key invalid:
Verify key at https://console.anthropic.com/
Check for spaces before/after key

Python not found:
Reinstall Python from https://python.org
Check "Add Python to PATH"

Connection refused:
Check port 8000 is available
Verify firewall allows port 8000
Restart application
