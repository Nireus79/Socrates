### Communication

All inter-library communication follows hub-and-spoke pattern:

```
Library A ──┐
Library B ──┼──> SocraticLibraryManager ──> AgentOrchestrator
Library C ──┘
     ↓
  No direct library-to-library communication
  All go through central manager
  Prevents circular dependencies
```
