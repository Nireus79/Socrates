# Archive Directory

This directory contains old files that are no longer actively used but kept for reference.

## Subdirectories:

- **old_docs/**: Old test output reports, analysis reports, and checklists
- **build_logs/**: Build pipeline outputs and deployment logs
- **scripts_temp/**: Temporary utility scripts
- **logs/**: Application logs and execution records

All files in this archive can be safely deleted if disk space is needed.
